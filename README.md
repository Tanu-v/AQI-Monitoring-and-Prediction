# Air Quality Index (AQI) Monitoring & Prediction

## Overview
This project focuses on predicting Air Quality Index (AQI) using machine learning techniques. The solution analyzes environmental data and forecasts AQI levels to support air quality monitoring.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- AQI Prediction Model
- Model Evaluation
- Data Visualization

## Results
- Built a Random Forest-based AQI prediction model.
- Achieved high prediction accuracy on environmental datasets.
- Developed a scalable framework for future real-time integration.
